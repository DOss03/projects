Bootstrap

Download und Installation

Bootstrap kann auf getbootstrap.com unter dem Button "Download" und nochmals einem Button "Download" heruntergeladen werden. Auf Visual Studio Code wird Bootstrap installiert, indem man die gedownloadete zip Datei extrahiert und in ihr dann auf derselben Ebene wie die Ordner css und js die .html und .css Dateien für die Website erstellt.

Grid-System von Bootstrap

Ein Grid-System besteht aus 12 Reihen und 12 Spalten. Man kann diese Reihen und Spalten beliebig auf sogenannte Container aufteilen.
